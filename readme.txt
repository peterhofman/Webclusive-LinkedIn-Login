== Installation ==
* Upload to the plugins directory.
* Apply for a API key on LinkedIn : https://www.linkedin.com/secure/developer?newapp=
* Add the Api key and secret in the admin screen.
* Add <?php webclusive_login_loginbutton(); ?> to add the login button.
* You are done!
